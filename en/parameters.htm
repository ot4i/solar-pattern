Parameters for the Solar pattern

Position
Pattern parameter Default Description
Places Hursley

Places

Use this drop down to select one of several predefined places. The Latitude and Longitude parameters each have an XPath expression that will extract and use the appropriate part of the drop down's value. Note that the enumeration value field only accepts alphanumerics and underscore
Latitude If you have selected Custom Places from the Places drop down enter the Latitude for the location you wish to find the sunrise and sunset details for.
Longitude If you have selected Custom Places from the Places drop down enter the Longitude for the location you wish to find the sunrise and sunset details for.
Queue information
Pattern parameter Default Description
Input queue name SOLAR

The queue where input messages are taken from.
Results queue name SUNSETSUNRISETIMES

The queue where the results are written too.
Error queue name SOLARERRORS

The queue where errors are written too.

All the pattern parameters are listed with their default values. Where no default is indicated, the default value is an empty string.
